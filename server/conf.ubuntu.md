### zsh
#### Установка
```Shell
sudo aptitude install zsh
chsh -s /bin/zsh
exit
```
После первого логина будет автоматически запущен __zsh-newuser-install__.
Нажать `0` (просто создастся файл .zshrc в домашней директории).
